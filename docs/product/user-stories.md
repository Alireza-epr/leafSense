## Problem Statement
Enable users to monitor the health and growth of vegetation in a selected area by viewing a de-clouded NDVI/vegetation index time series.

## User Stories

1. **As a farmer**, I can select my field on the map and view the vegetation health over time.
2. **As a user**, I can receive alerts when a selected area shows significant decline in vegetation index.
3. **As a researcher**, I can use the exported data.
4. **As a buyer of land**, I can analyze the vegetation health trends of a property, so I can make informed purchase decisions based on land quality.  

## Success Metrics

- Fast initial map loading 
- Pleasure response time
- Fast and smooth time series loading
- Select and compare multiple regions
- Export functionality
